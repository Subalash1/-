# 🎮 中文猜词游戏 - 启动说明

## 快速启动

### 1. 检查环境
确保已安装Python和Flask：
```bash
python --version
pip list | grep Flask
```

### 2. 启动游戏

#### 方式一：使用启动脚本（推荐）
**Windows用户**：
```bash
# 双击运行或命令行执行
start.bat
```

**Linux/Mac用户**：
```bash
chmod +x start.sh
./start.sh
```

#### 方式二：手动启动
```bash
# 激活虚拟环境（Windows）
activate_venv.bat

# 或手动安装依赖并启动
pip install flask pypinyin
python app.py
```

### 3. 访问游戏
打开浏览器访问：`http://localhost:5000`

## 🎯 游戏特色

✅ **美观的Web界面** - 现代化设计，渐变背景
✅ **字符颜色变化** - 字符本身变色，无颜色块
✅ **三维比较** - 汉字、拼音、声调同时比较
✅ **响应式设计** - 支持电脑、手机、平板
✅ **动画效果** - 流畅的界面动画
✅ **智能提示** - 实时反馈和错误处理

## 🎨 颜色系统

- 🟢 **绿色文字** = 正确位置
- 🟡 **黄色文字** = 存在但位置错误
- ⚫ **灰色文字** = 不存在

## 📱 使用说明

1. 点击"开始游戏"
2. 输入四字词语
3. 点击"提交"或按回车
4. 观察颜色反馈
5. 根据提示继续猜测
6. 最多6次机会猜中

## 🔧 如果遇到问题

### Flask无法启动
```bash
pip install flask
```

### pypinyin错误
使用独立版：
```bash
python app_standalone.py
```

### 端口被占用
修改代码中的端口号或关闭其他程序

## 📂 项目文件

- `app.py` - 主应用
- `start.bat` - Windows启动脚本
- `start.sh` - Linux/Mac启动脚本
- `activate_venv.bat` - 激活虚拟环境脚本
- `venv/` - Python虚拟环境
- `templates/index.html` - 网页模板
- `static/css/style.css` - 样式文件
- `static/js/script.js` - 交互逻辑

## 🎉 开始游戏吧！

选择一种启动方式，然后在浏览器中享受这个美观的中文猜词游戏！
